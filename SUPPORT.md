# Support

如果你想提问、反馈问题或参与协作，建议先按下面的入口分流。

## 使用问题与方案讨论

优先使用 GitHub Discussions：

- 适合提使用问题、部署思路、产品方案和架构讨论
- 适合“不确定是不是 bug”的问题

讨论入口：

- [GitHub Discussions](https://github.com/86Kolton/campus-smart-service/discussions)

## 功能缺陷与改进建议

请使用 GitHub Issues：

- 复现明确的功能缺陷
- 明确的功能增强或体验改进建议
- 文档错误、示例失效、脚本问题

提交前建议先看：

- [README.md](README.md)
- [docs/PROJECT_BRIEF.md](docs/PROJECT_BRIEF.md)
- [CONTRIBUTING.md](CONTRIBUTING.md)

## 安全问题

如果你发现的是安全漏洞，而不是普通功能缺陷，请不要直接公开发 Issue。

- 先阅读 [SECURITY.md](SECURITY.md)
- 按私下披露方式描述影响范围、复现条件和修复建议

## 暂不支持的内容

- 请求真实生产环境凭据
- 请求生产数据库、私有备份和私有恢复资料
- 请求公开仓库之外的内部部署细节

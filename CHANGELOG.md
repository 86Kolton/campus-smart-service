## [v0.1.0] - 2026-04-11

### Added

- Public GitHub repository for the project
- Open-source README with architecture, quick start, and repository boundaries
- MIT license, contributing guide, and security policy
- GitHub visual hero asset for repository presentation
- GitHub Actions CI workflow for backend tests and smoke checks
- Issue forms for bug reports and feature requests
- Pull request template
- GitHub release note category configuration

### Included in this initial public release

- Web user client built with HTML, CSS, and JavaScript
- FastAPI backend with admin APIs, client APIs, auth, and service modules
- RAG knowledge-base pipeline with document parsing, chunking, retrieval, and QA wiring
- Admin Studio static frontend for operations and configuration
- WeChat Mini Program client
- Backend regression tests and E2E scripts

### Notes

- The public repository intentionally excludes production secrets, private recovery assets, upload data, and runtime databases.

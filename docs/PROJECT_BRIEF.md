# 校园智服平台公开版项目概览

这个文档基于毕业设计分析材料整理为公开仓库版本，只保留对开源协作和项目理解有价值的内容，不包含学生个人信息、生产密钥或私有运维资料。

## 项目定位

校园智服平台是一个面向高校场景的“校园综合服务 + AI 智能问答”混合型系统。

- 学生用户通过 Web 用户端或微信小程序使用社区、问答、查询和跑腿等功能
- 管理员通过后台工作台进行知识库管理、文档上传、审核追踪、配置和日志查看
- 多端共享同一套 FastAPI 后端与核心业务服务

## 面向场景

- 校园社区互动与内容流转
- 校园知识库问答与办事指南检索
- 教务信息查询与消息提醒
- 校园跑腿与任务协作
- 知识库维护、自进化回灌与后台运营

## 核心模块

| 模块 | 关键能力 |
| --- | --- |
| 校园社区 | 信息流、发布、评论、点赞、收藏、搜索、AI 已采纳标识 |
| 知识库问答 | RAG 问答、意图识别、混合检索、重排、引用溯源 |
| 教务服务 | 成绩、课表、考试安排、空教室等查询能力 |
| 校园跑腿 | 任务发布、接单、送达、确认的状态闭环 |
| 消息中心 | 点赞、收藏、系统通知与未读计数 |
| 管理后台 | 知识库 CRUD、文档上传、自进化管理、配置与日志 |
| 知识自进化 | 候选帖子筛选、AI 审核、去重、向量化入库、追溯记录 |

## 技术架构

| 层级 | 方案 |
| --- | --- |
| 用户入口 | Web 用户端、微信小程序、管理后台 Studio |
| 后端 | Python 3.11+、FastAPI、SQLAlchemy、JWT、httpx |
| 数据层 | SQLite / PostgreSQL、Qdrant、Redis |
| AI / RAG | 硅基流动 API、Qwen/Qwen3-8B、BAAI/bge-m3、Qwen3-Reranker-4B |
| 部署层 | Nginx 反向代理、容器化部署、自动化测试与回归脚本 |

## 机制亮点

### 1. 社区驱动的知识库自进化

高质量帖子可经过筛选、审核、去重、向量化等处理流程进入知识库，形成“社区内容反哺智能问答”的闭环。

### 2. 隐私与智能双轨路由

敏感教务类查询优先走本地业务链路，校园经验类问答使用检索增强生成，兼顾可用性与数据边界。

### 3. 混合检索与查询扩展

系统结合向量检索、关键词召回、查询改写和重排策略，提升复杂校园问答的召回率和答案质量。

### 4. 多端共享统一后端

Web 用户端、微信小程序和管理后台共享同一套 API、鉴权、业务模型与服务能力，降低重复建设成本。

## 公开仓库边界

公开仓库保留：

- 源代码
- 示例知识库数据
- 测试脚本与 GitHub 协作配置
- 项目说明、版本记录和展示资源

公开仓库不保留：

- 真实环境变量与生产凭据
- 运行中的数据库与上传文件
- 私有运维恢复资料
- 管理员访问凭据和服务器敏感信息

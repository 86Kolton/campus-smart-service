# 社区行为准则

为了让这个公开仓库保持可协作、可讨论、可持续维护的状态，参与者默认遵守以下约定。

## 我们鼓励的行为

- 使用尊重、清晰、专业的表达
- 聚焦问题本身，给出可验证的反馈、复现路径和改进建议
- 在讨论设计、实现和架构时，允许不同观点，但要求论据充分
- 在提交 Issue、Discussion 或 Pull Request 前，先阅读现有文档与模板
- 主动保护项目中的安全边界、数据边界和隐私边界

## 不接受的行为

- 人身攻击、骚扰、羞辱、歧视性言论
- 恶意刷屏、无意义争吵、引战或破坏协作秩序
- 公开索取或传播生产环境账号、真实密钥、数据库和私有恢复资料
- 上传包含真实学生隐私、个人敏感信息或未脱敏业务数据的内容
- 明知是安全问题却公开披露可直接利用的细节

## 协作边界

- 功能建议优先走 Issue 或 Discussions
- 安全问题优先走 [SECURITY.md](SECURITY.md) 中的私下披露流程
- 如果改动涉及 UI、交互、文档结构或协作流程，请尽量给出上下文和验证方式

## 维护者处理方式

维护者可以根据实际情况删除不合适内容、关闭不合适讨论、拒绝不合适贡献，必要时限制相关账号的进一步互动。

## 适用范围

本准则适用于本仓库中的 Issue、Discussions、Pull Request、代码评论以及与项目协作直接相关的公开沟通场景。
